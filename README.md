# CogniFactory

A comprehensive platform for cognitive task management and automation.

## Overview

CogniFactory is designed to streamline cognitive workflows and enhance productivity through intelligent task processing and management.

## Features

- Task automation and scheduling
- Workflow management
- Real-time monitoring
- User-friendly interface
- RESTful API

## Installation

1. Clone the repository:
```bash
git clone https://github.com/chhaibiiheb/CogniFactory.git
cd CogniFactory
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Configure environment variables:
```bash
cp .env.example .env
```

## Usage

Start the application:
```bash
python app.py
```

Access the application at `http://localhost:5000`
